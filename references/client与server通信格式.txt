流程简介（省略包中换行，包的格式严格按上面来）
夜晚
先群发一个Night x 包
先汇报一下场上情况//reportall 一个Status包
关掉所有开关 //closeall Set 0 0
守卫（如果活着，下略）：
问守卫你守谁//askforonevote("Guard") 发一个Guard包 等10s（client实现等10s和判断不能连续守同一个人，下略）返回一个Vote包
给30s，狼人的狼人聊天频道打开，同时投票也打开，选择干谁//发一个Set 1 1 Wolf包，wolfchat包随便发，要求收到所有活着狼人的vote包，时间到后再发一个Set 0 0 Room包
女巫阶段：
给10s，发一个Witch @user包，告知女巫谁死了，返回一个vote包（不救@后面留空，救就把那个名字原样写上，client判断只有初夜可以自救）
给10s，发一个Witch包，返回一个vote包
预言家：
给10s，发一个Prophet包，返回一个vote包，发一个Good/Bad包

白天
先群发一个Day x 包
关掉所有开关
群发昨晚谁死了//发Die包，如果没人死就一个@也没有
如果你是猎人而且死了，10s时间，发给你一个Carry包，发一个Set 1 0 Room 返回一个vote包选择带谁走（@后面留空表示不带人，请client根据上一次的status和上一个Die包告诉李二不能带已死的人）时间到发一个Set 0 0 Room
如果李二死了而且选择带人，群发一个Hunter包，不需要任何回复
如果你是警长而且死了，群发Captaindie，发给你一个Pass包，Set 1 0 Room 10s时间，回一个vote包，然后Set 0 0 Room ，群发Captain
对初夜死的人或者猎人带走的人，10s时间，发一个Tellsth包，一个Set 0 1 Room 返回一个roomchat包，时间到发一个Set 0 0 Room
报告一下情况
第一日：
以下均对活着的玩家
群发一个 Choosecap包，然后挨个发set 0 1 Room包，每人10s说句话（roomchat包），不选就说不选，时间到了发一个Set 0 0 Room包
10s时间投票，发一个Set 1 0 Room包，返回vote包，时间到了发一个Set 0 0 Room包
群发一个Captian @包，不需要回复
以下每日相同
10s，发给警长（如果有）一个Order包，Set 1 0 Room 返回一个vote @1/0 然后Set 0 0 Room
按顺序活着的人15s Set 0 1 Room 返回一个roomchat 然后Set 0 0 Room
警长（如果存在） 20s Set 0 1 Room 返回一个roomchat 然后Set 0 0 Room
群发：Vote包 10s Set 1 0 Room 返回一个vote包然后 Set 0 0 Room
如果平票 发Equal包，重复一次上述投票(从新发一遍Vote包）（请client限制只能投Equal里面的人）

就跟上面初夜一样发Die包，按猎人什么的结算，都有遗言，结算完后报告一次情况
