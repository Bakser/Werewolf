join roomnumber

build roomnumber     //向用户发送你是房主后才要求一个build包指定房间类型 换行一定要标准的\n
xx xx xx xx xx xx xx     //共几个玩家 几个狼 几个预言家 几个女巫 几个守卫 几个猎人 几个警长（0/1）行尾最好多一个空格

leave roomnumber

up roomnumber

down roomnumber

start roomnumber

worldchat    //gamestart之后不允许世界频道，建立room后仍然可以
xxxxxxxxx@username//爱说啥说啥,按格式带上自己的用户名

roomchat roomnumber //这部分我会带着报头原样转发给各个能看到的user，世界频道和房间频道的显示以及说话人ID的显示请client做
xxxxxxxx@username    //RoomHandler做了广播，但也会转发给Game  
//
wolfchat roomnumber //夜晚狼人频道
xxxxxxxxx@username 

vote roomnumber
@username //投/指定的那个人的username username中不允许带换行符,若@后\n前没有东西表示弃票
